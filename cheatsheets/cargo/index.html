Cargo

Cargo is a package manager for Rust. It comes with most Rust installations. It manages downloading packages and dependecies, compiles your packages and can be used to upload it to package registries.

Basics

To create a new package and create the related directory use:

cargo new directory_name

To build your binary inside your package’s /target/debug/<package_name> directory run:

cargo build

If you want to build your binary with optimizations for release run:

cargo build --release

To compile and run your binary use:

cargo run
